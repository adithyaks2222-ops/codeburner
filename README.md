# 🔥 CodeBurner AI

CodeBurner AI is a savage developer assistant that roasts your code while giving real technical insights.

## 🚀 Features

* 🔥 Funny AI code roast
* 🧠 Smart technical insights
* 💡 Actionable suggestions
* 🎯 Code quality score
* 🧬 Developer type detection
* ⚡ Auto roast on errors

## 🛠️ Requirements

* VS Code
* Ollama installed and running

Run:

```
ollama run llama3
```

## ▶️ How to Use

1. Open any code file
2. Select code (or leave empty for full file)
3. Run command:

```
CodeBurner: Burn My Code
```

## 🤖 Powered By

* Ollama (Local AI)
* LLaMA 3 Model

## ⚠️ Notes

* Make sure Ollama is running locally
* Works best with JavaScript, Python, C, C++, Java
